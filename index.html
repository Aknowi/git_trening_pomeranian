Git podstawy

Mój cheat sheet

Komenda Opis
git init - inicjalizacja repozytorium
git status - status
git branch -m master main - zmiana nazwy brancha
git add index.html - włączenie śledzenia pliku (staging/paczka)
git status -s - skrot statusu, troche inny wyglad
git commit -m "podstawowa struktura dokumentu html" - zakomitowanie/zapisanie zmian pod określoną nazwą
A index.html index śledzony
M index.html index zmodyfikowany, należy ponownie "git add"
